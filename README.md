# H264 Player

A websocket h264 player written in React.

> forked from https://github.com/udevbe/ffmpeg-h264-wasm.git

## Getting Start

### start 

```
yarn
yarn run start
```

### build

```
yarn run build
```


## Fixed Bugs

- rendered green screen on different dimensions
- green edge when use ```gl.pixelStorei(gl.UNPACK_ALIGNMENT, 1)```;
